## Dev Tools > Deploy > Release Notes

### 2026. 04. 14.
#### 機能追加
* API v2.1ガイド新規追加
    * User Access Keyトークン(Bearer)による認証方式を適用しました。
* 情報照会APIを追加しました。
    * API v2.0及びv2.1において、アーティファクト一覧の照会、サーバーグループ一覧の照会、バイナリグループ一覧の照会、デプロイ履歴の照会、バイナリ一覧の照会の各APIを提供します。

### 2025. 04. 29.
#### 機能追加
* デプロイ実行API Version 2.0追加
    * Auto Scaleサーバーグループにもデプロイを実行できるようになりました。

#### 機能改善
* Cloud AgentアーティファクトのCommand TypeでもAuto Scaleサーバーグループを作成できるように変更  
* APIでバイナリをアップロードする際の容量制限を2GBから4GBに変更
